# Project Docs

This folder holds repository notes and generated planning artifacts that are not active tool instruction files.

- `docs/pr-description.md`: generated PR description artifact.
- `docs/review-handoff.md`: review handoff covering parent and nested V1 app changes.
- `docs/skill-builder-prompt.md`: prompt and edge-case notes used to create the PR-description skill workflow.

Root-level `AGENTS.md` and `CLAUDE.md` stay at the repository root because local tools use those paths as active instructions.
